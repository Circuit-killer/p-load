Instructions for building the official Pololu installer for p-load

These instructions assume you have already installed the proper version of
Visual Studio to build the project.

1) Download and install the WiX Toolset:

  http://wixtoolset.org/

2) Download and install the Windows SDK (for signtool.exe):

  http://msdn.microsoft.com/en-us/windows/hardware/hh852363.aspx

3) Make sure you have the Pololu Corporation signing certificate installed on
   your computer.

4) Make sure you have added setup_banner_wix.bmp and setup_welcome_wix.bmp to
   this directory.

5) In a Command Prompt, navigate to this directory and run build_installer.bat.
